# React - Chatbot Flow Builder

> Flow builder for Chatbot


Live version: [chatflowbuilder.vercel.app](https://chatflowbuilder.vercel.app)

### Scripts

- Run dev server
```bash
pnpm dev
```
- Run test suite
```bash
pnpm test
```
- Run E2E tests
```bash
pnpm test:e2e
```
- Run E2E tests in UI mode
```bash
pnpm test:e2e:ui
```
- Run E2E tests codegen
```bash
pnpm test:e2e:gen
```
